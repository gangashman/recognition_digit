# recognition_digit


Dependencies
-----------

Install PyBrain 
```bash
git clone git://github.com/pybrain/pybrain.git
python setup.py install
```

Other:
```bash
pip install scipy
pip install mnist
pip install python-mnist
pip install matplotlib
```
